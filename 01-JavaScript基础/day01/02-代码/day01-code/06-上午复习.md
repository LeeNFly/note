9 .变量的几种用法
1. 只声明
var p ; 
- 打印 undefined
- 说明还没有写完; 
- 结构 : var + 变量名

2. 先声明,再赋值
var p;  //声明
p = 12;   // 赋值  = 以后看见一个 = 等号,就是赋值  

3. 声明 + 赋值
var p = 21;

4. 声明多个变量并赋值
var a , b ;  // 声明多个变量
var c = 10, d = 20; // 声明c和d这个变量,并且赋值10 和 20

// 打印连写
console.log(c,d);

5. 不声明,,直接赋值 (不用 )
num =  21;

6. 不声明,也不赋值,,直接使用
console.log(aaaaa);   ==> 报错
